input type="number" click down instead of up demo

Instructions
  1. Reload window
  2. Do not hover on any elements
  3. Unfocus this window
  4. Aim at the invisible up button
  5. Click once to focus the window
  6. Without moving, click again
  7. Confirm you have clicked the up button
  8. Note it decremented
  9. You can keep decrementing until you move the cursor

Bug occurs at all font-sizes.

It seems like once you have hovered over the up/down buttons the bug is no long reproduciable until reload. However hovering over just the input (and not the up/down buttons), then unfocusing the window still allows the bug to be reproduced.
